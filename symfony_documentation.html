Install Symfony
Below Command install the Symfony in our system using Composer.
Command: composer create-project symfony/website-skeleton my_project_name
  1. Go to the created project folder "cd my_project_name" and then Run the Symfony Application use below command.
    Command: php -S 127.0.0.1:8000 -t public
  2. Install the annotation inside the symfony project use the below command in the terminal.
    Command: composer require annotations.
Install Symfony
Below Command used in the Symfony.
  1. Start the Symfony Appilcation.
    Command: php -S 127.0.0.1:8000 -t public
  2. List of console command.
    Command: php bin/console
  3. Check Router path for symfony application.
    Command: php bin/console debug:router
  4. Add the server namespace default command.
    Command: composer require server --dev

Databases, Doctrine, Fixtures

  1. Install the Doctrine in our symfony application using the below command.
    Command: composer require symfony/orm-pack
  2. Now install the maker-bundle in the development environment.
    Command: composer require symfony/maker-bundle --dev
  3. After install the doctrine bundle we need to configure the database credentials in the .env file.
    DATABASE_URL="mysql://USERNAME:PASSWORD@127.0.0.1:3306/dbname?serverVersion=5.7"
    to
    DATABASE_URL="mysql://root:@127.0.0.1:3306/Blogs?serverVersion=mariadb-10.4.13"
  4. Create the database using the Below command. "it will create database name as we mentioned in .env file"
    Command: php bin/console doctrine:database:create
  5. Create the table and entity file for the table using below command.
    Command: php bin/console make:entity
  6. After that we need to generate the migration process.

    Command: php bin/console make:migration = this used to create the migration file inside the migration folder.

    >Command: php bin/console doctrine:migrations:migrate = this command is used to create the table in the table by using migration file

Add the Project In GIT
  1. After install the Symfony appilcation. we need to add the symfony application in Git hub by using below commands
    Command: git clone (it is used to initiate the git repository URL)
  2. Check the status of Git hub.
    Command: git status
  3. add the project by using below command in git hub.
    Command: git add .( . stand for every thing need to commit)
  4. To commit the application use the below command in git hub.
    Command: git commit -m "init" (m stand for message)

Route the Symfony path to Symfony page

  1. Create New ".twig" file inside the template folder(project/templates/filename.html.twig)
  2. Add the below code in routes.yaml (project/config/routes.yaml)
    filename is hello_page hello_page: path: / controller: Symfony\Bundle\FrameworkBundle\Controller\TemplateController::templateAction defaults: template: 'hello_page.html.twig'
  3. Run the the command to check the Router is set or not line.
    php bin/console debug:router
